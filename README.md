# FortNexus – The Ultimate Worldwide Fortnite Hub 2026

A complete Next.js 15 Fortnite fan website with dark mode, glassmorphism design, neon colors, and full animations.

## Tech Stack
- Next.js 15 App Router + TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Features
- Live streams section
- Daily item shop with modal
- Streamers directory with search/filter
- News & leaks feed with tabs
- Global stats & leaderboard
- Upcoming events with countdown timers
- Community gallery
- Scroll progress bar, back-to-top button
- Particle background animation
- Fully responsive & mobile-friendly

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
